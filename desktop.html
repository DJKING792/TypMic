手机连不上？点这里排查
  • ① 同一网络：手机和电脑必须在同一个 WiFi，或电脑连手机热点。
  • ② 安卓：首次打开提示证书风险，点「高级 → 继续访问」即可。
  • ③ iPhone：iOS 无「继续访问」，须把电脑项目根目录下的 rootCA.pem 用 AirDrop / 邮件 / 微信文件传到 iPhone 安装并开启完全信任（设置 › 通用 › 关于本机 › 证书信任设置），再打开页面。
  • ④ 拒绝连接 / ERR_CONNECTION_REFUSED？ 多为 Windows 防火墙拦截。右键项目里的 allow_firewall.bat → 以管理员身份运行（只需一次）即可放行 8443。
